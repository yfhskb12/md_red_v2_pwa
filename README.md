
# Markdown Redactor 2.0

A high-performance Markdown editor built with React.

## Features

- **Split View Editor**: Real-time markdown preview.
- **Multi-Format Support**: Create and edit Markdown (.md), JSON (.json), and LookML (.lkml) files.
- **Syntax Highlighting**: Support for code blocks and specialized highlighting for LookML.
- **File Management**: Create folders, projects, and manage documents with drag-and-drop.
- **Export Options**: Export to Markdown, HTML, RTF, or print to PDF.

## Getting Started

1.  Clone the repository.
2.  Install dependencies (if applicable for your build system).
3.  Run the application.

## Technologies

- React
- Tailwind CSS
- Lucide React (Icons)
- Marked.js & DOMPurify
- Highlight.js
